COMO USAR NO RENDER.COM

1. Crie um repositório no GitHub com esses arquivos.
2. No Render.com, clique em 'New Web Service', conecte ao GitHub e selecione este repositório.
3. Escolha:
   - Environment: Docker
   - Build Command: (em branco)
   - Start Command: (em branco)
4. Configure seu banco MySQL e edite os arquivos PHP com os dados corretos.

Pronto! Sua loja estará online via Docker + Render.
